DPW Geofeed

Geographic IP feed data mapping IP addresses to locations.

Files

  • geofeed.csv - The complete geofeed database

Format

CSV format with the following columns:

  • CIDR - IP address range in CIDR notation
  • Country - Country code (ISO 3166-1 alpha-2)
  • Region - Region/state code
  • City - City name
  • Postal Code - Postal code
